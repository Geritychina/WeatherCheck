# 🌤 Weather Check

Simple React application for showing city weather forecast, using **free API** от [wttr.in](https://wttr.in).

---

## 🔍 Demo

 Type City name and press „Check“, for receive:
- Temperature
- Description
- Humidity

---

## 🧰 Used Technology

- ⚛️ React
- 🎨 CSS 
- 🌐 [wttr.in API](https://wttr.in/:help) — no API key needed.
